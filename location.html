Introduction to HTML by Matt

HTML

stands for Hyper Text Markup Language

DOCTYPE

declares which version of HTML is being followed by the document in question. Each version of HTML has its own set of rules, exclusions and inclusions. If an HTML document follows the rules specified in a particular version of HTML, then it is called valid HTML document. It is always a good practice to design valid HTML pages.

HTML element represents the main content of an HTML page

TITLE sets a title for an HTML document.

example of the code

working example

Subdivision of a written composition A paragraph is a subdivision of a written composition that consists of one or more sentences, deals with one point or gives the words of one speaker, and begins on a new usually indented line

div

The "div" tag is a fundamental element in HTML used to define divisions or sections within a web page. It acts as a container for other HTML elements, allowing developers to group and style them collectively using CSS or manipulate them with JavaScript.

descriptive list

The "dl" HTML element represents a description list. The element encloses a list of groups of terms (specified using the "dt" element) and descriptions (provided by "dd" elements). Common uses for this element are to implement a glossary or to display metadata

example of the code

<dl>
<dt>Mario</dt>
<dd>the red one</dd>
<dt>Luigi</dt>
<dd>the green one</dd>
</dl>

working example

Mario
Is the red character
Luigi
is the green character

Quote

The "q" element is used for shorter, inline quotations. It automatically adds quotation marks around the text.

working example

"As Albert Einstein once said, Imagination is more important than knowledge."

example of the code

<p>As Albert Einstein once said, <q>Imagination is more important than knowledge.</q></p>

blockquote

 

The blockquote element is used for longer quotations that are usually displayed as a block of text. It often includes an optional cite attribute to reference the source of the quote. "The only limit to our realization of tomorrow is our doubts of today."
- Franklin D. Roosevelt

example of the code

<blockquote>
"The only limit to our realization of tomorrow is our doubts of today."
<footer>- Franklin D. Roosevelt</footer>
</blockquote>

abbreviations

The abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Tip: Use the global title attribute to show the description for the abbreviation/acronym when you mouse over the element.

working example

The WHO was founded in 1948.

example of the code

<p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.</p>
